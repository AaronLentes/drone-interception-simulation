# Drone Interception Simulation

## Overview

**This project is currently in progress and will be available starting April 13.** 


This project presents a simulation-based analysis of autonomous drone interception strategies derived from scientific literature. It combines semi-autonomous systematic literature analysis with model implementation and experimental evaluation to identify the key factors governing interception success.

---

## Objectives

* Identify relevant interception strategies from scientific literature using a structured search approach
* Implement and validate a simulation model based on a selected reference paper
* Quantify the influence of key parameters on interception success through controlled experiments
* Demonstrate a reproducible workflow linking literature analysis, simulation, and experimental evaluation

---

## Methodology

The project follows a structured, evidence-driven workflow:

* Development of an initial search strategy based on the PICO framework
* Forward and backward snowballing to identify relevant studies
* Prioritization of literature using bibliometric analysis

  * **Content metrics:** keyword co-occurrence
  * **Citation-based network metrics:** citation velocity and citation chaining
* Definition of a reproducible stopping criterion to ensure search space convergence
* Selection of a central reference paper for model implementation

---

## Simulation Model

* Implementation of a pursuit–evasion scenario involving autonomous drones
* Simulation developed in Python
* Reproduction of the scenario described in the reference publication
* Validation of model behavior against expected system dynamics

---

## Experimental Design

A structured Design of Experiments (DoE) was conducted to analyze the sensitivity of interception performance to key system parameters.

Typical factors investigated include:

* Relative speed between interceptor and target
* Detection or reaction delay
* Initial distance and geometry
* Guidance or control parameters
